# Simple Shooter (Unreal Engine 5)

## Overview
Simple Shooter is a small Unreal Engine project created to explore **core gameplay programming concepts** such as player input, shooting mechanics, basic AI behavior, and Unreal Engine C++ workflows.

This project serves as a **learning and experimentation project** rather than a full systems-focused architecture.

---

## Features
- Basic player shooting mechanics
- Simple weapon logic
- Basic enemy AI
- Health and damage interaction

---

## Purpose of This Project
- Learn Unreal Engine C++ fundamentals
- Understand the gameplay framework (Actor, Pawn, Character)
- Experiment with basic AI and shooting mechanics

---

## Scope & Limitations
- Single-player focused
- Limited system abstraction
- Not designed for reuse or scalability

---

## Tech Stack
- Unreal Engine 5
- C++
- Blueprints (for rapid iteration)

---

## Notes
This project represents an **early-stage learning step** and is intentionally kept simple.
More advanced and reusable systems can be found in the **AdvancedShooter** project.
